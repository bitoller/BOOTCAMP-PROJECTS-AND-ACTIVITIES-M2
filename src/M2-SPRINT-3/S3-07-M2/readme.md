CSS | Atividade - Media queries e estilizando textos

Introdução
Nas atividades a seguir você pode criar os arquivos, utilizar o CodePen ou outra plataforma. Cada exercício possui um link para o CodePen, com uma estrutura base. Vamos lá, hora de praticar.

Katas 1 - Aprendendo a utilizar Media Queries
Link do katas 1.

1- A partir de 480px mude a cor de fundo do body para #8A2BE2, mude a cor da fonte para #fff e aumente a fonte

2- A partir de 767px mude a cor de fundo do body para #FF7F50 e a cor da fonte para #A9A9A9

3- Entre 768px e 979px mude a cor de fundo do body para #000 e a cor da fonte para #7FFF00

4- Defina o título com tamanho Grande.(30px).

5- Para desktop grande com mínimo de 1200px atribua a cor #E0FFFF de fundo para body e #000 para a cor de fonte

Katas 2 - Conceito de breakpoints com media query
Link do katas 2.

1- Altere o código para que a cor de fundo do body e dos textos mude à medida que a tela seja redimensionada. Depois redimensione a tela para observar na prática.

2- Altere a cor de fundo do body para salmão(#AF5A5A) e a cor dos textos para branco(#FFFFFF) a partir da resolução 768px;

3- Altere novamente a cor de fundo do body para azul-claro(#5A73AF);

4- Altere novamente a cor do body, agora para verde água(#5AAF9C);

Note que a cor dos textos continua sendo branco. Obs: as cores informadas são apenas sugestões, você pode utilizar outras.
